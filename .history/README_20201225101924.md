一个小Demo
可以获取选中html中的图片
在火狐浏览器中选中内容区域,右键点击---查看选中部分源代码，然后把html内容放到多行文本框中，点击替换按钮，页面会展示出来相应的内容（没有css样式），再点击发送就下载图片啦
本Demo由于是为了方便修改阿里巴巴商品详情图中的尺寸，所有会替换---.32x32为空，要使用的话根据实际情况进行选择
